Sie können Naia OS sofort ausprobieren — schließen Sie einfach ein USB-Laufwerk an und schalten Sie Ihren PC ein. Keine Installation erforderlich, und Ihr bestehendes System bleibt unberührt.

![USB anschließen und KI ausführen](/plug-usb-and-run-ai.webp)

## Was Sie brauchen

- Ein USB-Laufwerk (8 GB+) mit darauf geschriebener Naia OS
- Jeder PC, der in den letzten 10 Jahren hergestellt wurde

> Sie haben kein USB-Laufwerk bereit? Siehe **[3.1. Naia OS Installation (ISO)](/de/manual/install-iso)**, um zu erfahren, wie Sie eines erstellen.

## So verwenden Sie es

1. Schließen Sie das Naia OS USB-Laufwerk an Ihren PC an.
2. Schalten Sie ein (oder starten Sie neu) und rufen Sie das Boot-Menü auf (drücken Sie `F12`, `F2` oder `Entf` wiederholt direkt nach dem Einschalten).
3. Wählen Sie das USB-Laufwerk als Boot-Gerät aus.
4. Naia OS startet automatisch.

## Was Sie sehen werden

![Naia OS Desktop vom USB-Laufwerk](/images/manual/iso-install/01-live-desktop-naia.png)

Der Naia OS Desktop erscheint und **die Naia App startet automatisch**.

![Naia App läuft](/images/manual/iso-install/12-naia-app.png)

Wählen Sie Ihren KI-Anbieter und beginnen Sie sofort mit Naia zu chatten. Erkunden Sie den Desktop und probieren Sie alle Funktionen aus — es funktioniert genau wie ein vollständig installiertes System.

> **Wi-Fi-Verbindung erforderlich.** Lokale KI wie Ollama kann in einer Live-USB-Umgebung nicht installiert werden, daher benötigen Sie einen Cloud-KI-Anbieter. Verbinden Sie sich zuerst über die Systemleiste unten rechts auf dem Bildschirm mit Wi-Fi.

> **Nichts wird auf Ihrem PC gespeichert.** Wenn Sie das USB-Laufwerk entfernen und neu starten, ist Ihr ursprüngliches System wieder genau so, wie es war. Fühlen Sie sich frei zu experimentieren!

> **Hinweis:** In der Live-USB-Umgebung werden alle Einstellungen beim Neustart zurückgesetzt. Discord-Kanalintegration, Skill-Konfigurationen und der Konversationsverlauf werden nicht gespeichert. Um diese Funktionen wirklich zu nutzen, installieren Sie Naia OS auf Ihrem PC.

## Möchten Sie es behalten?

Wenn Ihnen die Erfahrung gefällt und Sie Naia OS dauerhaft installieren möchten, siehe **[3.1. Naia OS Installation (ISO)](/de/manual/install-iso)**. Nach der Installation können Sie auch Ollama einrichten, um Ihre eigene private KI zu haben, die nicht von einem Cloud-Dienst abhängt.

Sie können auch nur die Naia App auf Ihrem bestehenden Linux installieren — siehe **[3.2. Linux App Installation](/de/manual/install-app)**.
