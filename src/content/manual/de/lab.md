## API Keys

![API key management](lab-keys.png)

Verwalten Sie API Keys für Desktop-Apps und externe Dienste:
- **Neuen Key erstellen**: Geben Sie einen Key-Namen ein und generieren Sie ihn
- **Key löschen**: Entfernen Sie ungenutzte Keys
- Keys werden automatisch ausgegeben, wenn Sie sich von der Desktop-App verbinden
