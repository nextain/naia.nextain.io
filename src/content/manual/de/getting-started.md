Wenn Sie die App zum ersten Mal starten, erscheint ein Onboarding-Assistent. Es gibt zwei Möglichkeiten, zu beginnen.

## Mit Naia starten (Empfohlen)

Der einfachste Weg – kostenlos und kein API Key erforderlich.

![Onboarding-Bildschirm – Anbieterauswahl](onboarding-provider.png)

1.  Klicken Sie auf die Schaltfläche **Naia** auf dem Onboarding-Bildschirm
2.  Ein Browser öffnet sich zur Login-Seite naia.nextain.io
3.  Melden Sie sich mit Ihrem **Google account** an (die Registrierung erfolgt automatisch)
4.  Sie werden automatisch zurück zur Desktop-App weitergeleitet
5.  Sie erhalten **500 kostenlose Credits** bei der Anmeldung

## Mit API Key starten

Sie können auch einen API Key von Anbietern wie Google Gemini eingeben.

![Onboarding-Bildschirm – API Key-Eingabe](onboarding-apikey.png)

1.  Wählen Sie einen Anbieter aus (Gemini usw.)
2.  Klicken Sie auf **Weiter**
3.  Geben Sie Ihren API Key ein
4.  Klicken Sie auf **Validieren**, um den Key zu testen

## Onboarding-Schritte

Nach der Naia-Anmeldung oder API Key-Eingabe schließen Sie diese Schritte ab:

### 1. Benennen Sie Ihre KI

![KI-Namenseingabe](onboarding-agent-name.png)

Geben Sie Ihrem KI-Avatar einen beliebigen Namen (z.B. "Naia", "Luna", "Cookie").

### 2. Ihr Name

![Ihr Name Eingabe](onboarding-user-name.png)

Geben Sie ein, wie Naia Sie nennen soll.

### 3. Charakter auswählen

![Charakterauswahl-Bildschirm](onboarding-character.png)

Wählen Sie aus 4 VRM-Avataren:

| Charakter       | Beschreibung           |
|-----------------|------------------------|
| Shino           | Schuluniform (Standard)|
| Sakurada Fumiriya | Lässig               |
| Girl            | Generisches Mädchen    |
| Boy             | Generischer Junge      |


### 4. Persönlichkeit auswählen

![Persönlichkeitsauswahl-Bildschirm](onboarding-personality.png)

Wählen Sie aus 4 Voreinstellungen:

| Persönlichkeit | Beschreibung                  |
|----------------|-------------------------------|
| Freundlicher Begleiter | Lockere Sprache, warme Persönlichkeit |
| Zuverlässiger Assistent | Formelle Sprache, professioneller Ton |
| Spaßvogel | Humorvoll, fröhlicher Ton |
| Ruhiger Gelehrter | Intellektueller, besonnener Ton |


### 5. Sprachstil wählen

Wählen Sie, wie Ihre KI spricht:

| Stil | Beschreibung |
|------|--------------|
| Lässig | Wie ein Freund |
| Formell | Höflich und respektvoll |

### 6. Abschließen

![Onboarding abgeschlossen Bildschirm](onboarding-complete.png)

Sie können auf diesem Bildschirm optional einen **Discord-Bot** verbinden.
Klicken Sie auf **Legen Sie los**, um zum Hauptbildschirm zu gelangen.
