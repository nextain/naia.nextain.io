Khi bạn lần đầu khởi chạy ứng dụng, một trình hướng dẫn chào mừng sẽ xuất hiện. Có hai cách để bắt đầu.

## Bắt đầu với Naia (Được khuyến nghị)

Cách dễ nhất — miễn phí và không cần API key.

![Onboarding screen — provider selection](onboarding-provider.png)

1. Nhấp vào nút **Naia** trên màn hình chào mừng
2. Trình duyệt sẽ mở trang đăng nhập naia.nextain.io
3. Đăng nhập bằng **tài khoản Google** của bạn (đăng ký tự động)
4. Bạn sẽ tự động được chuyển hướng trở lại ứng dụng máy tính để bàn
5. Bạn nhận được **500 tín chỉ miễn phí** khi đăng ký

## Bắt đầu với API Key

Bạn cũng có thể nhập API key từ các nhà cung cấp như Google Gemini.

![Onboarding screen — API key entry](onboarding-apikey.png)

1. Chọn một nhà cung cấp (Gemini, v.v.)
2. Nhấp **Tiếp theo**
3. Nhập API key của bạn
4. Nhấp **Xác thực** để kiểm tra key

## Các bước chào mừng

Sau khi đăng nhập Naia hoặc nhập API key, hoàn thành các bước sau:

### 1. Đặt tên cho AI của bạn

![AI name input](onboarding-agent-name.png)

Đặt tên cho Naia bất kỳ tên nào bạn muốn (ví dụ: "Naia", "Luna", "Cookie").

### 2. Tên của bạn

![Your name input](onboarding-user-name.png)

Nhập tên mà Naia nên gọi bạn.

### 3. Chọn nhân vật

![Character selection screen](onboarding-character.png)

Chọn từ 4 hình đại diện VRM:

| Nhân vật | Mô tả |
|-----------|-------------|
| Shino | Đồng phục học sinh (mặc định) |
| Sakurada Fumiriya | Thường ngày |
| Girl | Cô gái chung |
| Boy | Chàng trai chung |


### 4. Chọn tính cách

![Personality selection screen](onboarding-personality.png)

Chọn từ 4 cài đặt sẵn:

| Tính cách | Mô tả |
|-------------|-------------|
| Bạn thân thiện | Nói chuyện thân mật, tính cách ấm áp |
| Trợ lý đáng tin cậy | Nói chuyện trang trọng, giọng chuyên nghiệp |
| Tinh nghịch | Hài hước, giọng vui tươi |
| Học giả điềm tĩnh | Giọng trí thức, điềm tĩnh |


### 5. Chọn phong cách nói

Chọn cách AI của bạn nói chuyện:

| Phong cách | Mô tả |
|------------|-------|
| Thân mật | Như bạn bè |
| Trang trọng | Lịch sự và tôn trọng |

### 6. Hoàn tất

![Onboarding complete screen](onboarding-complete.png)

Bạn có thể tùy chọn kết nối **bot Discord** từ màn hình này.
Nhấp **Bắt đầu** để vào màn hình chính.
