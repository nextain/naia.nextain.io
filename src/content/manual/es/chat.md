La pantalla principal de conversación con Naia.

## Chat de Texto

![Chat screen](chat-text.png)

1. Escriba un mensaje en el campo de entrada en la parte inferior
2. Presione **Enter** o haga clic en el **botón Enviar**
3. Naia genera una respuesta (transmite en tiempo real)
4. Use Shift+Enter para saltos de línea

## Chat de Voz (en vivo)

1. Haga clic en el **botón de Chat de Voz** junto al campo de entrada
2. Una vez conectado, comienza una conversación de voz bidireccional en tiempo real
3. Hable y Naia responde inmediatamente con voz
4. Haga clic de nuevo para finalizar la sesión de voz

- La entrada de texto sigue disponible durante el chat de voz
- El avatar realiza animación de sincronización labial durante las respuestas

## Visualización de Costos

- **Costo total de API** para la sesión actual mostrado en la esquina superior derecha
- Haga clic para abrir el panel de control de costos detallado:
  - Mensajes por proveedor
  - Recuentos de tokens de entrada/salida
  - Saldo de Naia (cuando esté conectado)
  - Enlace de recarga de crédito

## Nueva Conversación

- Haga clic en el botón **+** (esquina superior derecha) para iniciar una nueva conversación
- Las conversaciones anteriores se guardan en la pestaña Historial

> **Memoria**: Naia recuerda naturalmente información sobre ti durante la conversación — tu nombre, preferencias, cumpleaños y más. Estos recuerdos persisten en todos los canales, incluyendo Discord DM. No se necesita ninguna acción manual; solo chatea y los recuerdos se acumulan automáticamente. Todos los datos se almacenan localmente en tu dispositivo.

## Visualización de Ejecución de Herramientas

Cuando Naia utiliza herramientas, el área de chat muestra:

- Nombre de la herramienta (p. ej., "Leer Archivo", "Ejecutar Comando")
- Estado de ejecución (ejecutándose / éxito / error)
- Resultados (expandibles)

## Modal de Aprobación

Para ejecuciones de herramientas de alta seguridad:

| Botón | Descripción |
|--------|-------------|
| **Permitir una vez** | Permite esta única ejecución |
| **Permitir siempre** | Aprobar automáticamente esta herramienta en el futuro |
| **Rechazar** | Denegar la ejecución |
