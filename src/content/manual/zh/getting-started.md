首次启动应用时，会出现一个引导向导。有两种方式可以开始使用。

## 从 Naia 开始（推荐）

最简单的方式 — 免费且无需 API Key。

![Onboarding screen — provider selection](onboarding-provider.png)

1. 在引导屏幕上点击 **Naia** 按钮
2. 浏览器将打开 naia.nextain.io 登录页面
3. 使用 **Google** 或 **Discord** 帐户登录（注册是自动的）
4. 您将自动重定向回桌面应用
5. 注册后您将获得 **500 个免费积分**

一个Naia账户即可使用多种LLM提供商、实时语音对话、多轮对话、设置备份与同步等。详情请参阅 **[16. Naia账户](/zh/manual/naia-account)**。

## 从 API Key 开始

您也可以输入来自 Google Gemini 等提供商的 API Key。

![Onboarding screen — API key entry](onboarding-apikey.png)

1. 选择一个提供商 (Gemini, 等)
2. 点击 **下一步**
3. 输入您的 API Key
4. 点击 **验证** 来测试 Key

## 引导步骤

完成 Naia 登录或 API Key 输入后，请完成以下步骤：

### 1. 为您的 AI 命名

![AI name input](onboarding-agent-name.png)

为AI助手命名，任何您喜欢的名字都可以（例如，”Naia”、”Luna”、”Cookie”）。

### 2. 您的名字

![Your name input](onboarding-user-name.png)

输入 Naia 应该如何称呼您。

### 3. 选择角色

![Character selection screen](onboarding-character.png)

从 4 个 VRM 头像中选择：

| 角色 | 描述 |
|---|---|
| Shino | 校服（默认） |
| Sakurada Fumiriya | 休闲装 |
| Girl | 普通女孩 |
| Boy | 普通男孩 |


### 4. 选择个性

![Personality selection screen](onboarding-personality.png)

从 4 个预设中选择：

| 个性 | 描述 |
|---|---|
| 亲切的朋友 | 随意说话，温暖的性格 |
| 可靠的助手 | 正式用语，专业语气 |
| 调皮鬼 | 幽默，开朗的语气 |
| 沉稳的学者 | 知性沉稳的语调 |


### 5. 选择说话风格

选择AI的说话方式：

| 风格 | 描述 |
|------|------|
| 随意 | 像朋友一样 |
| 正式 | 礼貌而尊重 |

### 6. 完成

![Onboarding complete screen](onboarding-complete.png)

您可以在此屏幕上选择连接 **Discord 机器人**。
点击 **开始使用** 进入主屏幕。
