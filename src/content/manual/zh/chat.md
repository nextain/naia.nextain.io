与AI助手的主对话屏幕。

## 文本聊天

![聊天屏幕](chat-text.png)

1.  在底部的输入框中输入消息
2.  按下 **Enter** 键或点击 **发送按钮**
3.  AI助手生成回复（实时流式传输）
4.  使用 Shift+Enter 进行换行

## 语音聊天（实时）

1.  点击输入框旁边的 **语音聊天按钮**
2.  连接完成后，实时双向语音对话开始
3.  说话后AI会立即以语音回应
4.  再次点击结束语音会话

-   语音聊天期间仍可使用文本输入
-   AI头像在回应时会进行唇形同步动画

## 费用显示

-   当前会话的**总 API 费用**显示在右上角
-   点击以打开详细费用仪表盘：
    -   每个提供商的消息数
    -   输入/输出令牌计数
    -   Naia余额（连接时）
    -   充值链接

## 新对话

-   点击右上角的 **+** 按钮开始新对话
-   之前的对话保存在”历史记录”选项卡中

> **记忆功能**：Naia会在对话过程中自然记住关于您的信息（姓名、偏好、生日等）。这些记忆在所有渠道（包括Discord私信）中持续有效。无需手动操作，只需正常聊天，记忆就会自动积累。所有记忆仅存储在您的本地设备上。

## 工具执行显示

当AI助手使用工具时，聊天区域显示：

-   工具名称（例如，“读取文件”，“执行命令”）
-   执行状态（运行中 / 成功 / 错误）
-   结果（可展开）

## 审批模态框

对于高安全性工具执行：

| 按钮         | 描述             |
| ------------ | ---------------- |
| **允许一次** | 允许本次执行     |
| **始终允许** | 此后自动审批此工具 |
| **拒绝**     | 拒绝执行         |
