## 사용량

![사용량 차트](lab-usage.png)

일별 차트로 사용량을 확인합니다:
- 요청 수
- 토큰 수
- 비용
- 기간 선택: 7일 / 30일 / 90일

## 로그

![로그 테이블](lab-logs.png)

상세 API 호출 로그를 확인합니다:
- 상태, 모델별 필터링
- 시간, 상태, 모델, 토큰, 비용 확인
- 각 행을 클릭하면 상세 정보 표시

## API 키

![API 키 관리](lab-keys.png)

데스크톱 앱이나 외부 서비스에서 사용할 API 키를 관리합니다:
- **새 키 생성**: 키 이름을 입력하고 생성
- **키 삭제**: 사용하지 않는 키 제거
- 데스크톱 앱 연결 시 자동으로 키가 발급됩니다

## 연동 (Integrations)

naia.nextain.io 계정과 외부 메신저를 연동하여 어디서든 AI 아바타와 대화할 수 있습니다.

### Discord
- Discord 계정으로 로그인하면 자동 연동
- Naia 봇에게 **DM**으로 대화
- 크레딧은 Naia 계정에서 자동 차감

### Google Chat
- Google Chat 연동은 준비 중입니다 (곧 지원 예정)
