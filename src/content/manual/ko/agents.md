Naia에서 동작하는 여러 에이전트(메인 AI 아바타 포함)와 서브 에이전트들을 관리하는 탭입니다.

![에이전트 탭](agents-tab.png)

## 에이전트 목록
- 현재 실행 중인 모든 에이전트 카드 목록을 볼 수 있습니다.
- 새로운 복잡한 작업이 시작되면, AI 아바타가 백그라운드에서 서브 에이전트를 생성하여 작업을 병렬로 처리합니다.

## 파일 관리
에이전트 카드 하단의 **Files** 버튼을 누르면 해당 에이전트의 로컬 작업 공간에 생성된 파일 목록이 나타납니다.

- **파일 읽기**: 목록에서 파일을 클릭하면, 텍스트 영역에 파일의 내용이 표시됩니다.
- **파일 수정**: 텍스트 영역에서 내용을 수정한 뒤 **저장** 버튼을 누르면 즉시 파일이 업데이트됩니다.

## 세션 관리
현재 에이전트들의 대화 세션도 함께 관리할 수 있습니다.
- **삭제(Delete)**: 불필요한 세션을 삭제합니다.
- **압축(Compact)**: 너무 길어진 세션 기록을 요약하여 시스템 메모리를 확보합니다.
