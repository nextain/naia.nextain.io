앱을 처음 실행하면 온보딩 마법사가 나타납니다. 두 가지 방법으로 시작할 수 있습니다.

## Naia으로 시작하기 (추천)

가장 쉬운 방법입니다. API 키 없이 무료로 시작할 수 있습니다.

![온보딩 화면 — 제공자 선택](onboarding-provider.png)

1. 온보딩 화면에서 **Naia** 버튼을 클릭합니다
2. 브라우저가 naia.nextain.io 로그인 페이지로 열립니다
3. **Google 계정**으로 로그인합니다 (자동 회원가입)
4. 자동으로 데스크톱 앱으로 돌아옵니다
5. 가입 시 **무료 5 크레딧**을 받습니다

## API 키로 시작하기

LLM 제공업체의 API 키를 직접 입력할 수도 있습니다.

![온보딩 화면 — API 키 입력](onboarding-apikey.png)

1. 제공업체를 선택합니다 (Claude Code CLI, Gemini, OpenAI, Anthropic, xAI, zAI, Ollama)
2. **다음**을 클릭합니다
3. API 키를 입력합니다
4. **검증** 버튼으로 키를 테스트합니다

## 온보딩 단계

Naia 로그인 또는 API 키 입력 후, 다음 단계를 진행합니다:

### 1. AI 이름 짓기
AI 아바타에게 원하는 이름을 지어주세요. (예: "AI 아바타", "루나", "쿠키")

![AI 이름 입력](onboarding-agent-name.png)

### 2. 내 이름 입력
AI 아바타가 나를 부를 이름을 입력합니다.

![내 이름 입력](onboarding-user-name.png)

### 3. 캐릭터 선택
4가지 VRM 아바타 중 하나를 고릅니다:

| 캐릭터 | 설명 |
|--------|------|
| Shino | 유니폼 (기본) |
| Sakurada Fumiriya | 캐주얼 |
| Girl | 일반 여성 |
| Boy | 일반 남성 |

![캐릭터 선택 화면](onboarding-character.png)

### 4. 성격 선택
4가지 성격 프리셋 중 하나를 고릅니다:

| 성격 | 설명 |
|------|------|
| 다정한 친구 (Friendly) | 따뜻하고 다정한 말투 |
| 듬직한 비서 (Polite) | 정중하고 격식 있는 말투 |
| 장난꾸러기 (Playful) | 유머러스하고 밝은 말투 |
| 차분한 학자 (Calm) | 차분하고 지적인 말투 |

![성격 선택 화면](onboarding-personality.png)

### 5. 말투 선택
반말과 존댓말 중 하나를 선택합니다:

| 말투 | 설명 |
|------|------|
| 반말 | 편한 친구처럼 캐주얼한 말투 |
| 존댓말 | 정중하고 격식 있는 말투 |

### 6. 완료

![온보딩 완료 화면](onboarding-complete.png)
**시작하기** 버튼을 클릭하면 메인 화면으로 이동합니다.
