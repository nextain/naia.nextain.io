Главный экран чата с Naia.

## Текстовый чат

![Chat screen](chat-text.png)

1. Введите сообщение в поле ввода внизу
2. Нажмите **Enter** или кнопку **Отправить**
3. Naia генерирует ответ (поступает в реальном времени)
4. Используйте Shift+Enter для переноса строки

## Голосовой разговор

1. Нажмите кнопку **микрофона** рядом с полем ввода
2. Говорите, когда микрофон активируется
3. Нажмите еще раз, чтобы остановить запись — речь преобразуется в текст (STT)
4. Преобразованный текст отправляется автоматически

## Голосовой ответ (TTS)

- Ответы Naia автоматически воспроизводятся в виде аудио
- Можно отключить в Настройках
- Аватар выполняет синхронизацию губ во время воспроизведения

## Отображение стоимости

- **Общая стоимость API** для текущей сессии отображается в правом верхнем углу
- Нажмите, чтобы открыть подробную панель стоимости:
  - Сообщения по провайдерам
  - Количество входящих/исходящих токенов
  - Баланс Naia (при подключении)
  - Ссылка для пополнения кредита

## Новый диалог

- Нажмите кнопку **+** (вверху справа), чтобы начать новый диалог
- Предыдущие диалоги сохраняются во вкладке История

> **Память**: Naia естественным образом запоминает информацию о вас во время беседы — ваше имя, предпочтения, день рождения и многое другое. Эти воспоминания сохраняются во всех каналах, включая Discord DM. Никаких ручных действий не требуется — просто общайтесь, и память накапливается автоматически. Все данные хранятся только на вашем устройстве.

## Отображение выполнения инструмента

Когда Naia использует инструменты, в области чата отображается:

- Имя инструмента (напр., «Прочитать файл», «Выполнить команду»)
- Статус выполнения (выполняется / успешно / ошибка)
- Результаты (разворачиваемые)

## Модальное окно подтверждения

Для выполнения высокозащищенных инструментов:

| Кнопка | Описание |
|--------|-------------|
| **Разрешить один раз** | Разрешить это единственное выполнение |
| **Всегда разрешать** | Автоматически одобрять этот инструмент в дальнейшем |
| **Отклонить** | Отклонить выполнение |
