При первом запуске приложения появляется мастер адаптации. Есть два способа начать.

## Начните с Naia (Рекомендуется)

Самый простой способ — бесплатно и без необходимости в API Key.

![Onboarding screen — provider selection](onboarding-provider.png)

1. Нажмите кнопку **Naia** на экране адаптации.
2. Откроется браузер на странице входа naia.nextain.io.
3. Войдите с помощью вашей **учетной записи Google** (регистрация происходит автоматически).
4. Вы будете автоматически перенаправлены обратно в настольное приложение.
5. При регистрации вы получаете **500 бесплатных кредитов**.

## Начните с API Key

Вы также можете ввести API Key от таких поставщиков, как Google Gemini.

![Onboarding screen — API key entry](onboarding-apikey.png)

1. Выберите поставщика (Gemini и т.д.)
2. Нажмите **Next**
3. Введите ваш API Key
4. Нажмите **Validate**, чтобы протестировать ключ

## Шаги адаптации

После входа в Naia или ввода API Key выполните следующие шаги:

### 1. Назовите свой ИИ

![AI name input](onboarding-agent-name.png)

Дайте Naia любое имя, какое захотите (например, "Naia", "Luna", "Cookie").

### 2. Ваше имя

![Your name input](onboarding-user-name.png)

Введите, как Naia должна вас называть.

### 3. Выберите персонажа

![Character selection screen](onboarding-character.png)

Выберите из 4 VRM аватаров:

| Персонаж | Описание |
|-----------|-------------|
| Shino (Темный) | Темная униформа |
| Shino (Светлый) | Светлая униформа |
| Девочка | Обычная девочка |
| Мальчик | Обычный мальчик |


### 4. Выберите личность

![Personality selection screen](onboarding-personality.png)

Выберите из 4 предустановок:

| Личность | Описание |
|-------------|-------------|
| Дружелюбный | Непринужденный, теплый тон |
| Профессиональный | Вежливый, формальный тон |
| Игривый | Юмористический, яркий тон |
| Ученый | Спокойный, интеллектуальный тон |


### 5. Интеграция с мессенджерами (Необязательно)

![Messenger integration screen](onboarding-messenger.png)

Настройте интеграцию с поддерживаемыми мессенджерами, такими как Telegram или Slack. Этот шаг можно пропустить и настроить позже в настройках.

### 6. Завершение

![Onboarding complete screen](onboarding-complete.png)
Нажмите **Get Started**, чтобы перейти на главный экран.
