Naia может быть установлена двумя способами. Каждый из них является полностью независимым методом — выберите тот, который подходит вашей ситуации.

> Хотите попробовать перед установкой? **[2. Naia OS Live USB](/ru/manual/live-usb)** позволяет вам опробовать Naia прямо с USB-накопителя.

## Naia OS — Полная установка ОС (Рекомендуется)

Загрузитесь с USB-накопителя и установите напрямую на ваш жесткий диск. Naia глубоко интегрирована в ОС и запускается автоматически при каждой загрузке.

- **Что вам понадобится**: USB-накопитель (8 ГБ+), ПК и 64 ГБ+ дискового пространства.

**[Посмотреть руководство по установке Naia OS →](/ru/manual/install-iso)**

## Установка приложения для Linux

Установите Naia как отдельное приложение в вашей существующей системе Linux. Переустанавливать ОС не требуется.

- Поддерживает Flatpak, AppImage, DEB и RPM.

**[Посмотреть руководство по установке приложения →](/ru/manual/install-app)**
