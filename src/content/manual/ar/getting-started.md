عند تشغيل التطبيق للمرة الأولى، يظهر معالج الإعداد الأولي. هناك طريقتان للبدء.

## البدء باستخدام Naia (موصى به)

الطريقة الأسهل — مجانية ولا تتطلب مفتاح API.

![شاشة الإعداد الأولي — اختيار المزود](onboarding-provider.png)

1.  انقر على زر **Naia** في شاشة الإعداد الأولي
2.  يفتح متصفح على صفحة تسجيل الدخول naia.nextain.io
3.  سجّل الدخول باستخدام **حساب Google** الخاص بك (التسجيل تلقائي)
4.  سيتم إعادة توجيهك تلقائيًا مرة أخرى إلى تطبيق سطح المكتب
5.  تتلقى **20 رصيدًا مجانيًا** عند التسجيل

## البدء باستخدام مفتاح API

يمكنك أيضًا إدخال مفتاح API من مزودين مثل Google Gemini.

![شاشة الإعداد الأولي — إدخال مفتاح API](onboarding-apikey.png)

1.  اختر مزودًا (Gemini، إلخ.)
2.  انقر على **التالي**
3.  أدخل مفتاح API الخاص بك
4.  انقر على **التحقق** لاختبار المفتاح

## خطوات الإعداد الأولي

بعد تسجيل الدخول إلى Naia أو إدخال مفتاح API، أكمل هذه الخطوات:

### 1. تسمية الذكاء الاصطناعي الخاص بك

![إدخال اسم الذكاء الاصطناعي](onboarding-agent-name.png)

امنح Naia أي اسم ترغب به (مثل "Naia"، "Luna"، "Cookie").

### 2. اسمك

![إدخال اسمك](onboarding-user-name.png)

أدخل الاسم الذي يجب أن يناديك به Naia.

### 3. اختيار الصورة الرمزية

![شاشة اختيار الصورة الرمزية](onboarding-character.png)

اختر من بين 4 صور رمزية VRM:

| الصورة الرمزية | الوصف        |
|----------------|--------------|
| Shino (غامق)   | زي غامق      |
| Shino (فاتح)   | زي فاتح      |
| فتاة           | فتاة عامة    |
| صبي            | صبي عام      |


### 4. اختيار نمط الشخصية

![شاشة اختيار نمط الشخصية](onboarding-personality.png)

اختر من بين 4 إعدادات مسبقة:

| نمط الشخصية | الوصف                 |
|-------------|-----------------------|
| ودود        | نبرة عفوية ودافئة     |
| احترافي    | نبرة مهذبة ورسمية     |
| مرح         | نبرة فكاهية ومشرقة    |
| باحث        | نبرة هادئة وذكية      |


### 5. دمج المراسلة (اختياري)

![شاشة دمج المراسلة](onboarding-messenger.png)

قم بإعداد الدمج مع تطبيقات المراسلة المدعومة مثل Telegram أو Slack. يمكن تخطي هذه الخطوة وتكوينها لاحقًا في الإعدادات.

### 6. اكتمال

![شاشة اكتمال الإعداد الأولي](onboarding-complete.png)
انقر على **بدء الاستخدام** للدخول إلى الشاشة الرئيسية.
