アプリを初めて起動すると、オンボーディングウィザードが表示されます。開始するには2つの方法があります。

## Naiaで開始する（推奨）

最も簡単な方法 — 無料でAPI Keyは不要です。

![Onboarding screen — provider selection](onboarding-provider.png)

1. オンボーディング画面で**Naia** ボタンをクリックします
2. ブラウザがnaia.nextain.ioのログインページを開きます
3. **Google** または **Discord** アカウントでサインインします（登録は自動です）
4. デスクトップアプリに自動的にリダイレクトされます
5. サインアップ時に **500無料クレジット** を受け取ります

Naiaアカウント1つで、複数のLLMプロバイダー、ライブ音声会話、マルチターン会話、設定バックアップ・同期などが利用できます。詳しくは **[16. Naiaアカウント](/ja/manual/naia-account)** をご覧ください。

## API Keyで開始する

Google GeminiのようなプロバイダーからAPI Keyを入力することもできます。

![Onboarding screen — API key entry](onboarding-apikey.png)

1. プロバイダーを選択します（Geminiなど）
2. **次へ**をクリックします
3. API Keyを入力します
4. **検証**をクリックしてキーをテストします

## オンボーディング手順

NaiaログインまたはAPI Key入力後、以下の手順を完了してください。

### 1. AIに名前を付ける

![AI name input](onboarding-agent-name.png)

AIアバターに好きな名前を付けてください（例：「Naia」、「Luna」、「Cookie」）。

### 2. あなたの名前

![Your name input](onboarding-user-name.png)

Naiaに呼ばれたい名前を入力してください。

### 3. キャラクターを選択する

![Character selection screen](onboarding-character.png)

4つのVRMアバターから選択してください。

| キャラクター | 説明 |
|-----------|-------------|
| Shino | 制服（デフォルト） |
| Sakurada Fumiriya | カジュアル |
| Girl | 一般的な女の子 |
| Boy | 一般的な男の子 |


### 4. パーソナリティを選択する

![Personality selection screen](onboarding-personality.png)

4つのプリセットから選択してください。

| パーソナリティ | 説明 |
|-------------|-------------|
| 親しい友達 | 気楽なタメ口、温かい性格 |
| 頼れる秘書 | 敬語、丁寧な話し方 |
| いたずらっ子 | ユーモラスで明るい話し方 |
| 落ち着いた学者 | 知的で落ち着いたトーン |


### 5. 話し方を選ぶ

AIの話し方を選んでください：

| スタイル | 説明 |
|---------|------|
| タメ口 | 友達のように |
| 敬語 | 丁寧で礼儀正しい |

### 6. 完了

![Onboarding complete screen](onboarding-complete.png)

この画面から**Discordボット** を接続することもできます（任意）。
**はじめる**をクリックしてメイン画面に入ります。
