## Naia দিয়ে শুরু করুন (প্রস্তাবিত)

সবচেয়ে সহজ উপায় — বিনামূল্যে এবং কোনো API key প্রয়োজন নেই।

![Onboarding screen — provider selection](onboarding-provider.png)

1. অনবোর্ডিং স্ক্রিনে **Naia** বোতামে ক্লিক করুন
2. একটি ব্রাউজার naia.nextain.io লগইন পৃষ্ঠায় খোলে
3. আপনার **Google account** দিয়ে সাইন ইন করুন (নিবন্ধন স্বয়ংক্রিয়)
4. আপনাকে স্বয়ংক্রিয়ভাবে ডেস্কটপ অ্যাপে ফিরিয়ে আনা হবে
5. সাইন-আপ করার সময় আপনি **500টি বিনামূল্যে ক্রেডিট** পাবেন

## API Key দিয়ে শুরু করুন

আপনি Google Gemini-এর মতো প্রদানকারীদের থেকে একটি API key-ও প্রবেশ করাতে পারেন।

![Onboarding screen — API key entry](onboarding-apikey.png)

1. একটি প্রদানকারী নির্বাচন করুন (Gemini, ইত্যাদি)
2. **Next** ক্লিক করুন
3. আপনার API key প্রবেশ করান
4. কীটি পরীক্ষা করতে **Validate** ক্লিক করুন
