এই ট্যাবটি অ্যাপের সাথে সংযুক্ত বিভিন্ন মেসেজিং চ্যানেল (Discord, Slack, Google Chat, Telegram, ইত্যাদি) পরিচালনা করে।

![Channels Tab](channels-tab.png)

## চ্যানেল তালিকা দেখা
আপনি সংযুক্ত সমস্ত চ্যানেল এবং প্রতিটি অ্যাকাউন্টের স্থিতি এক নজরে দেখতে পারবেন।

- **স্থিতি ব্যাজ**: `connected`, `disconnected`, বা `error`-এর মতো অবস্থা প্রদর্শন করে।
- **রিফ্রেশ**: সর্বশেষ স্থিতি পেতে উপরের ডানদিকে রিফ্রেশ বোতামে ক্লিক করুন।

## চ্যানেল লগইন (QR কোড)
আপনি একটি নির্দিষ্ট চ্যানেল সংযোগ করতে একটি ওয়েব লগইন শুরু করতে পারেন।
চ্যাটে Naia-কে "Start Discord web login" বলে, এটি একটি QR কোড প্রদর্শন করবে অথবা প্রয়োজন অনুযায়ী একটি প্রমাণীকরণ অপেক্ষার অবস্থায় প্রবেশ করবে।

## Discord বট ইন্টিগ্রেশন (শুধুমাত্র DM)

আপনার অ্যাকাউন্ট স্বয়ংক্রিয়ভাবে লিঙ্ক করতে [naia.nextain.io](https://naia.nextain.io)-এ আপনার Discord অ্যাকাউন্ট দিয়ে সাইন ইন করুন। একবার লিঙ্ক হয়ে গেলে, আপনি Discord-এ DM (ব্যক্তিগত বার্তা) এর মাধ্যমে Naia-এর সাথে চ্যাট করতে পারবেন।

### কিভাবে ব্যবহার করবেন
1. naia.nextain.io-এ **Discord দিয়ে সাইন ইন করুন**
2. **Settings > Integrations** পৃষ্ঠায় বট ইনভাইট লিঙ্কে ক্লিক করুন
3. চ্যাট করতে **বটকে একটি DM পাঠান**
4. আপনার naia.nextain.io অ্যাকাউন্ট থেকে ক্রেডিট স্বয়ংক্রিয়ভাবে কেটে নেওয়া হবে

### বৈশিষ্ট্যসমূহ
- **শুধুমাত্র DM**: Naia ব্যক্তিগত বার্তায় (DM) প্রতিক্রিয়া জানায়
- **ক্রেডিট ইন্টিগ্রেশন**: আপনার naia.nextain.io অ্যাকাউন্ট থেকে ক্রেডিট স্বয়ংক্রিয়ভাবে ব্যবহার করা হয়
- **নিবন্ধনবিহীন ব্যবহারকারী নির্দেশিকা**: লিঙ্ক করা অ্যাকাউন্ট ছাড়া ব্যবহারকারীরা সেটআপ নির্দেশাবলী পান
- **রেট লিমিটিং**: আপনার ক্রেডিট সুরক্ষিত রাখতে প্রতি মিনিটে 10টি বার্তার সীমা

## Google Chat ইন্টিগ্রেশন (শীঘ্রই আসছে)

Google Chat ইন্টিগ্রেশন একটি ভবিষ্যতের আপডেটের জন্য পরিকল্পিত। আপডেটের জন্য অপেক্ষা করুন।

## মেসেঞ্জার নোটিফিকেশন (ওয়েববুকস)
Naia OpenClaw-এর শক্তিশালী চ্যানেল সিস্টেমের উত্তরাধিকারী।
**Settings > Tools > Webhooks** মেনুতে অথবা প্রাথমিক অনবোর্ডিং স্ক্রিনে আপনার Slack, Discord বা Google Chat Webhook URL প্রবেশ করানোর মাধ্যমে, Naia আপনাকে গুরুত্বপূর্ণ কাজের ফলাফল সহ বার্তা পাঠাতে পারবে।

> **টিপস:** "Let me know on Discord when this file backup is completely finished!"

## অ্যাডভান্সড: একটি 24/7 স্বায়ত্তশাসিত বট তৈরি করুন
টার্মিনাল কমান্ড টুল (`execute_command`) ব্যবহার করে, আপনি Naia-কে একটি 24/7 স্বায়ত্তশাসিত এজেন্টে পরিণত করতে পারবেন যা আপনার ডেস্কটপের বাইরেও Telegram বা Discord-এ থাকে।

চ্যাটে Naia-কে এইভাবে কমান্ড দিন:
> "My Telegram bot token is `1234:ABC...`. Run `openclaw channels add --channel telegram --token 1234:ABC...` to start my Telegram bot."

এখন, এমনকি আপনি ডেস্কটপ অ্যাপ বন্ধ করলেও, আপনি আপনার ফোনে Telegram এর মাধ্যমে পটভূমির OpenClaw Gateway এর মাধ্যমে Naia-এর সাথে চ্যাট করতে এবং যেকোনো সময় কাজ বরাদ্দ করতে পারবেন।
