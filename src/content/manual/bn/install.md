Naia দুই ভাবে ইনস্টল করা যেতে পারে। প্রতিটি সম্পূর্ণ স্বাধীন পদ্ধতি — আপনার পরিস্থিতির সাথে মানানসই একটি বেছে নিন।

> ইনস্টল করার আগে চেষ্টা করতে চান? **[2. Naia OS Live USB](/bn/manual/live-usb)** আপনাকে শুধুমাত্র একটি USB ড্রাইভ থেকে Naia ব্যবহার করার সুযোগ দেয়।

## Naia OS — সম্পূর্ণ OS ইনস্টলেশন (প্রস্তাবিত)

একটি USB ড্রাইভ থেকে বুট করুন এবং সরাসরি আপনার হার্ড ড্রাইভে ইনস্টল করুন। Naia OS এর সাথে গভীরভাবে একত্রিত এবং প্রতিবার বুট করার সময় স্বয়ংক্রিয়ভাবে চালু হয়।

- **আপনার যা প্রয়োজন**: একটি USB ড্রাইভ (8 GB+), একটি PC, এবং 64 GB+ ডিস্ক স্পেস।

**[Naia OS ইনস্টলেশন গাইড দেখুন →](/bn/manual/install-iso)**

## লিনাক্স অ্যাপ ইনস্টলেশন

আপনার বিদ্যমান লিনাক্স সিস্টেমে Naia একটি স্বতন্ত্র অ্যাপ হিসাবে ইনস্টল করুন। আপনার OS পুনরায় ইনস্টল করার প্রয়োজন নেই।

- Flatpak, AppImage, DEB, এবং RPM সমর্থন করে।
