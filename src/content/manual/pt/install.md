Naia pode ser instalado de duas formas. Cada um é um método completamente independente — escolha o que melhor se adapta à sua situação.

> Quer experimentar antes de instalar? **[2. Naia OS Live USB](/pt/manual/live-usb)** permite-lhe experimentar o Naia a partir de apenas uma pen USB.

## Naia OS — Instalação Completa do SO (Recomendado)

Arranque a partir de uma pen USB e instale diretamente no seu disco rígido. O Naia está profundamente integrado no SO e é iniciado automaticamente em cada arranque.

- **O que precisa**: Uma pen USB (8 GB+), um PC e 64 GB+ de espaço em disco.

**[Ver o Guia de Instalação do Naia OS →](/pt/manual/install-iso)**

## Instalação da Aplicação Linux

Instale o Naia como uma aplicação autónoma no seu sistema Linux existente. Não é necessário reinstalar o seu SO.

- Suporta Flatpak, AppImage, DEB e RPM.

**[Ver o Guia de Instalação da Aplicação →](/pt/manual/install-app)**
