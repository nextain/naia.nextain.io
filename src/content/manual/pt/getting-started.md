Ao iniciar o aplicativo pela primeira vez, um assistente de integração aparece. Existem duas maneiras de começar.

## Começar com Naia (Recomendado)

A maneira mais fácil — gratuita e sem necessidade de API Key.

![Onboarding screen — provider selection](onboarding-provider.png)

1. Clique no botão **Naia** na tela de integração
2. Um navegador se abre na página de login naia.nextain.io
3. Faça login com sua conta **Google** ou **Discord** (o registro é automático)
4. Você será automaticamente redirecionado de volta para o aplicativo de desktop
5. Você recebe **500 créditos gratuitos** ao se cadastrar

Com uma conta Naia, acesse vários provedores LLM, conversa por voz ao vivo, diálogo multi-turno, backup de configurações e mais. Detalhes em **[16. Conta Naia](/pt/manual/naia-account)**.

## Começar com API Key

Você também pode inserir uma API Key de provedores como Google Gemini.

![Onboarding screen — API key entry](onboarding-apikey.png)

1. Selecione um provedor (Gemini, etc.)
2. Clique em **Próximo**
3. Insira sua API Key
4. Clique em **Validar** para testar a chave

## Etapas de Integração

Após o login no Naia ou a inserção da API Key, complete estas etapas:

### 1. Dê um Nome à Sua IA

![AI name input](onboarding-agent-name.png)

Dê ao Naia o nome que desejar (por exemplo, "Naia", "Luna", "Cookie").

### 2. Seu Nome

![Your name input](onboarding-user-name.png)

Insira como o Naia deve chamá-lo(a).

### 3. Escolha um Personagem

![Character selection screen](onboarding-character.png)

Escolha entre 4 avatares VRM:

| Personagem | Descrição |
|-----------|-------------|
| Shino | Uniforme escolar (padrão) |
| Sakurada Fumiriya | Casual |
| Girl | Menina genérica |
| Boy | Menino genérico |


### 4. Escolha uma Personalidade

![Personality selection screen](onboarding-personality.png)

Escolha entre 4 predefinições:

| Personalidade | Descrição |
|-------------|-------------|
| Amigo carinhoso | Fala informal, personalidade calorosa |
| Assistente confiável | Fala formal, tom profissional |
| Brincalhão | Humorístico, tom alegre |
| Estudioso calmo | Tom intelectual e sereno |


### 5. Escolha o Estilo de Fala

Escolha como sua IA fala:

| Estilo | Descrição |
|--------|-----------|
| Informal | Como um amigo |
| Formal | Educado e respeitoso |

### 6. Concluir

![Onboarding complete screen](onboarding-complete.png)

Você pode opcionalmente conectar um **bot do Discord** nesta tela.
Clique em **Comece** para entrar na tela principal.
