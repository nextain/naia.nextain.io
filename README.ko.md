# Naia OS 홈페이지 (naia.nextain.io)

[English](README.md)

이곳은 [Naia OS](https://github.com/nextain/naia-os)의 공식 홈페이지 및 사용자 포털입니다.
Naia OS에 대한 소개, 사용자 매뉴얼, 블로그, 그리고 계정 관리 기능을 제공합니다.

## 주요 기능
- **랜딩 페이지**: Naia OS 소개 및 가격 정보.
- **사용자 포털**: 계정, 연동(예: Discord), API 키 관리.
- **블로그**: 기술 블로그 (Firebase 댓글, GA4 인기 포스팅, CC BY-NC-SA 4.0 라이선스).
- **관리자 도구**: 포스트 에디터, Git 배포, Dev.to/Reddit 크로스포스팅, 댓글 관리 (localhost 전용).
- **매뉴얼**: Naia OS 사용을 위한 단계별 가이드.
- **Discord 봇 연동**: Naia OS Discord 봇 연결 및 관리.

## 시작하기

```bash
npm install
npm run dev
```

브라우저에서 [http://localhost:3001](http://localhost:3001)을 열어 결과를 확인하세요.
